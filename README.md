## ⏱ Universal Task Timer – Chrome Extension (Live on Web Store)

A productivity-focused Chrome Extension designed to help **QA engineers and professionals track time spent on tasks and reports directly from the browser**.

### Features

- Start and stop task timer directly from the browser  
- Enter **employee name, report name, and date** for task tracking  
- Add **custom key-value parameters** for flexible logging  
- View previous logs with **total time summary**  
- **Export logs to CSV** for reporting and documentation  
- All data stored **locally in the browser** (privacy-focused)

[Chrome Web Store Link] https://chromewebstore.google.com/detail/jdiginnpejaojfjcmojblhmjjbophfgg?utm_source=item-share-cb

This extension helps professionals maintain **accurate task tracking and reporting during daily QA work**, improving efficiency and accountability.
